# Neev Draft Command

Create a new blueprint

## Usage

Run this command to create a new blueprint.

## Instructions

Create a new blueprint for this feature

1. Gather requirements for the new feature
2. Create a blueprint with intent, architecture, API spec, and security
3. Validate the blueprint against project principles
4. Save to .neev/blueprints/
