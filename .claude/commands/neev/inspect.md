# Neev Inspect Command

Analyze project structure and find gaps

## Usage

Run this command to analyze project structure and find gaps.

## Instructions

Analyze the project structure and show me any gaps

1. Check for drift between specs and code
2. Identify missing components or inconsistencies
3. Report gaps that need to be addressed
4. Suggest fixes where possible
